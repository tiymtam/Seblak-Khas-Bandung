# 🍜 Seblak Khas Bandung - Landing Page Promosi

Project ini adalah *landing page* interaktif yang dibuat untuk memenuhi tugas Ujian Akhir Semester (UAS) mata kuliah **E-Business**, Program Studi Sistem Informasi, Universitas Negeri Semarang (UNNES).

Project ini berfokus pada digitalisasi produk kuliner lokal (Seblak) menjadi produk modern dengan inovasi **Self-Heating Cup** dan fitur pemesanan *custom topping* secara digital.

## 🔗 Live Demo
Kamu bisa melihat hasil *deploy* dari *landing page* ini melalui tautan berikut:
**[👉 Kunjungi Seblak Khas Bandung Live]((https://seblak-khas-bandung.vercel.app/))** 

## ✨ Fitur Utama (Unique Value Proposition)
* **Hero Section Interaktif:** Menampilkan visual produk yang menggugah selera dengan copywriting persuasif ala *startup* kuliner.
* **Problem & Solution:** Mengedukasi calon pembeli mengenai kepraktisan inovasi *Self-Heating Cup* tanpa memerlukan kompor.
* **Custom Topping Feature:** Fitur digital interaktif di mana pengguna dapat memilih level kepedasan dan merakit isian seblak secara mandiri sebelum diarahkan ke WhatsApp untuk pemesanan.
* **Cara Penyajian (How to Use):** Panduan 3 langkah mudah penyajian produk.
* **Testimoni Pelanggan (Social Proof):** Menampilkan ulasan untuk membangun kepercayaan target pasar (Gen Z dan Mahasiswa).

## 🛠️ Teknologi yang Digunakan
* **HTML5:** Struktur semantik halaman web.
* **CSS3:** *Styling* murni (Vanilla CSS) tanpa *framework*, mencakup animasi *floating*, *hover effects*, dan tata letak *grid/flexbox* yang responsif (*mobile-friendly*).
* **Google Fonts:** Menggunakan perpaduan *font* `Libre Baskerville`, `Satisfy`, dan `Plus Jakarta Sans` untuk kesan artisan dan modern.
* **Vercel:** Platform *cloud deployment* untuk *hosting* gratis.
